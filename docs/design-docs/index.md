# Design Docs — Index

Architecture Decision Records (ADRs) and design documents for the Dubai Property 3D World Platform.

## Format

Each design doc should follow this template:

```markdown
# ADR-XXX: Title

**Status:** Proposed | Accepted | Deprecated | Superseded
**Date:** YYYY-MM-DD
**Author:** Name

## Context
What is the issue or decision we need to make?

## Decision
What did we decide and why?

## Consequences
What are the trade-offs and implications?
```
